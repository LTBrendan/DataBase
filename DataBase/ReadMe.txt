DataBase Project - Mathis Roger, Brendan Le Trionnaire

Ce projet consiste en une application permettant de se connecter en tant que d'utilisateur ou administrateur à des bases de données grâce au JDBC.

Pré-requis :

Afin de pouvoir utiliser cette application, il est nécessaire de posséder une connexion à une base de données Oracle SQLDeveloper.
Pour cela il est possible de télécharger Oracle 12c sur le site internet d'oracle : https://www.oracle.com/fr/corporate/features/database-12c/index.htm
